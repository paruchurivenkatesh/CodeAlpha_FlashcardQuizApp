# 📚 Project Functionality

The Flashcard Quiz App helps users learn and practice using digital flashcards.

Users can:
- View flashcard questions
- Flip cards to see answers
- Navigate through cards
- Add their own flashcards

The application also includes modern UI effects and animations for better user experience.
